2.4 - Modo de Segurança

O que é modo de segurança?

O modo de segurança é uma forma especial de inicialização do Windows, muito utilizada para fazer a identificação e solução de problemas relacionados a software e drivers. Nesse  modo são carregados apenas os arquivos necessários para a inicialização do Windows, e o PC inicia com a resolução gráfica em VGA (o que permite exibir em qualquer monitor).

Conheça melhor o modo de segurança do Windows 10 (Foto: Reprodução/Edivaldo Brito)

Na maioria dos casos, o modo de segurança impedirá que software ou drivers de hardwares problemáticos sejam carregados pelo sistema, tornando mais fácil encontrar qualquer problema e corrigi-lo. Normalmente, esse modo é utilizado quando o PC está inoperante e não se sabe o problema ou não pode resolvê-lo.

Tipos de modo de segurança

Apesar do nome parecer algo único, existem diferentes tipos de modo de segurança. Cada um oferece variados níveis de funcionalidade que podem ser usadas em diversas situações. No modo mínimo (modo de segurança mais restrito), nenhum software adicional ou driver é executado. Na maioria dos casos, é utilizado para a resolução de problemas.

Há também um modo de segurança que permite o uso da linha de comando. Ou seja, caso precise usar o prompt de comando para resolver o seu problema.

Quem tiver problemas com o Active Directory, também pode usar um modo de segurança especifico. Neste caso, o Windows desativa o Active Directory durante a inicialização no modo de segurança (é habilitado em outros modos). 

Infelizmente, nem tudo pode ser resolvido offline. Por isso, quando precisar de uma conexão de Internet ativa para solucionar um problema usando o modo de segurança, use o modo com suporte a rede.

Como acessar o modo de segurança no Windows 10

Se o Windows não estiver conseguindo inicializar ou deixa de funcionar várias vezes, irá oferecer, automaticamente, a opção para inicializar no modo de segurança. Mas há outras formas de inicializar manualmente o Windows 10. 

O mais comum é pressionar a tecla F8 repetidamente enquanto o Windows está inicializando. Porém, nem sempre funciona no Windows 10, já que o tempo de boot é muito rápido. Outra maneira de acessar o modo de segurança é pressionando e segurando a tecla “Shift”, enquanto clica na opção “Reiniciar” do Windows.

Reiniciando o Windows (Foto: Reprodução/Edivaldo Brito)

Isto irá reiniciar o PC e exibir as opções de inicialização, onde você poderá acessar a opção “Solução de problemas”, “Opções avançadas”, “Configurações de inicialização” e “Reiniciar”, para reiniciar o Windows 10 e acessar as opções de modo de segurança.

Configurações de inicialização do Windows 10 (Foto: Reprodução/Edivaldo Brito)

Identificando problemas usando o modo de segurança

Se depois de entrar no modo de segurança, o problema desaparecer, isso significa que é uma falha no software ou driver. Se mesmo usando o modo de segurança, o problema persistir, então a falha é com o hardware do PC, e você terá que levá-lo para um técnico.

Resolvendo problemas usando o modo de segurança

Conheça algumas soluções para resolver problemas enquanto estiver usando o modo de segurança: 

Se o PC tem um vírus que o programa antivírus não pode excluir, escaneei no modo de segurança. É possível executar uma verificação de vírus para eliminar um malware. A maioria dos malwares não irá funcionar no modo de segurança, o que torna mais fácil excluí-los. Mas o programa antivírus pode não funcionar nesse modo, então será preciso criar um disco de recuperação do antivírus e usá-lo para fazer a varredura do PC.
Se você acha que um software adicionado recentemente causou o problema, basta iniciar o sistema em modo de segurança e desinstalar a partir do Painel de Controle.
Se alguma mudança recente nas configurações ou em algum software for o problema, o modo de segurança lhe dará o poder de restaurar o PC para uma data anterior ou redefini-la.
Drivers de hardware também podem ser um problema. Neste caso, pressione as Windows + R e digite devmgmt.msc na caixa de diálogo “Executar”. Isso abrirá o Gerenciador de Dispositivos com todos os dispositivos e drivers listados. Se houver um problema com um driver, exibirá o ícone com um triângulo amarelo ao lado. Clique com o botão direito do mouse sobre o item, e no menu que aparece, escolha “Atualizar o software” para atualizá-lo. Se isso não resolver a falha, terá de desinstalar e reinstalar o driver, fazendo o download do driver no site do fabricante.
O Registro do Windows funciona bem no modo de segurança. Se você tem instruções para resolver o problema usando o registro, fique a vontade para usá-lo.

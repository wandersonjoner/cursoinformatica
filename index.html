Informática para o Trabalho

The Future é a mais nova Escola de Tecnologia. Norteando o  ensino em tecnologias da Informação desde a montagem e configuração de computadores à fases mais avançadas do projeto educacional como lógica de programação, Banco de Dados relacionais, Arquitetura REST, planilhas no Excel, Word e muito mais.

Linguagens de programação  de Back-End como Java e de Front-End como o Angular que é patrocinado pela Google e pela Microsoft. Com o intuito de capacitar jovens e adultos para o mercado de trabalho norteado a ética e o empreendedorismo para o crescimento profissional dos alunos.
    
        
       
Modo básico e avançado: Montagem e configuração de computadores, Ética e Empreendedorismo, Aplicações para Escritório com a Pasta Office (Excel, Word, Email Outlook);
Modo Front-End básico: HTML, CSS, JavaScript;
Modo Back-End básico: Lógica de Programação, Linguagem Java, Banco de dados;
Modo Intermediário Front-End: Angular 2/4...;
Modo Intermediário Back-End: REST com Java Spring Boot, Spring MVC, Spring DATA;
Modo Avançado Back-End Spring Security;
